# Changelog

All notable changes to `localization-gen-vue-adapter` are documented in this file.

## [0.0.1] - 2026-04-16

### Added

- Initial release of Vue runtime adapter
- `createLocalizationPlugin` Vue plugin and `useLocalization` composable
- Locale store bridge and lazy/SSR helper utilities
- Compatibility with manifests generated by `localization-gen-core`

### Changed

- Canonical helper naming: `translate`, `format`, `plural`, `gender`, `context`, `namespace`

### Removed

- Deprecated helper aliases: `t`, `ti`, `tp`, `tg`, `tc`, `interpolate`, `resolvePlural`, `resolveGender`, `resolveContext`
- Deprecated `ns(scope)` alias; use `namespace(scope)` instead
- Deprecated `createNamespace(scope)`; renamed to `namespace(scope)`
